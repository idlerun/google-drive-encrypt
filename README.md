

## Local Testing

Edit /etc/hosts and add

```
127.0.0.1 drive-encrypt.com
```

Run in dev mode with `HTTPS_PORT=443 npm run dev`

Hit https://drive-encrypt.com as normal (and accept self-signed cert)
